# 🌌 Dojutsu Awakening: Perfect Alignment 👁️✨

> Witness the ultimate fusion of **Mangekyo Sharingan** 🔴 and **Rinnegan** 🟣 in an interactive anime-inspired eye experience!  

This project is a fully **interactive web animation** that brings your favorite **Naruto Dojutsu** powers to life. Move your mouse, click, and watch the eyes follow your cursor while the chakra particles flow around you. Perfect for fans, developers, and animators who love **anime-style visual experiments**.

---

## 🚀 Features

### 👁️ Eye Animations
- **Mangekyo Sharingan**
  - Spinning tomoe with **glowing effect**
  - Heartbeat animation
  - Interactive power-up mode (click to flare)
- **Rinnegan**
  - Multiple concentric rings with **ripple animations**
  - Subtle glow effect
  - Responsive heartbeat animation

### 🖱️ Interactive Controls
- **Mouse tracking**
  - Eyes rotate in 3D perspective following your cursor
  - Iris movement inside sclera for natural realism
- **Power Mode**
  - Click anywhere → eyes widen, glow flares, particles speed up
  - Release click → return to normal

### 🌌 Chakra Particle System
- Two types of particles: red (Sharingan) and purple (Rinnegan)
- Particles move upwards, accelerated in power mode
- Dynamic **real-time animation** using HTML5 Canvas

### ✨ Micro-Expressions
- Natural blinking with random intervals
- Slight squinting when focusing near the center
- Glow intensity adjusts based on mouse focus

### 🖌️ Eye Details
- **Sclera**
  - Realistic shading and shadow for depth
  - Blood veins with gradient transparency
- **Iris**
  - Reflections for realism
  - Glow effects controlled by CSS variables
- **Pupil**
  - Centered, glows subtly with heartbeat
- **Tomoe**
  - Rotating elements for Mangekyo Sharingan
  - Smooth animations with CSS keyframes

## 💻 Technology Stack

| Layer | Technology | Purpose |
|-------|------------|---------|
| Frontend | HTML5 | Markup structure |
| Styling | TailwindCSS + Custom CSS | Eye shapes, glow, reflections, animations |
| Animation | CSS Keyframes | Spinning, heartbeat, squinting |
| Canvas | HTML5 Canvas API | Particle system (chakra flow) |
| JS Interactivity | Vanilla JS | Mouse tracking, blinking, power mode, iris movement |
